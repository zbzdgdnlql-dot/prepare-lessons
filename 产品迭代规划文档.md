# Français Pro 备课平台产品迭代规划文档 (v2.0 规划)

基于当前 MVP（最小可行性产品）的成功验证，我们即将进入精细化打磨阶段。本文档梳理了基于现有反馈的 3 个核心需求落实方案，并从一线小语种教师视角进行了深度的发散性头脑风暴，以指导下一阶段的产品迭代。

---

## 一、 核心痛点与微调需求落实方案

针对您提出的三个核心需求，它们不应仅仅是零散的按钮，而应该自然地融入现有的“三步走”工作流中。

### 1. 分类化备课包内容，让教师可选
* **痛点场景**：有时候老师只需要出几道随堂测验题，或者只需要文化背景介绍，不需要生成长篇大论的完整教案，全量生成既浪费 Token 又耗时。
* **产品落地点：修改【第二步：确认大纲与生成设置】**
  * **设计方案**：在第二步（大纲确认页面）底部，在“生成教案”按钮上方增加一个 **“备课模块配置（Checklist）”** 区域。
  * **可选模块示例**：
    * [x] 词汇精讲（含例句）
    * [x] 语法解析与易错点
    * [x] 课文文化背景/拓展阅读
    * [x] 课堂互动游戏/Activity设计
    * [x] 随堂小测验（单选、填空）
    * [x] 课后作业布置
  * **价值**：赋予教师生成控制权，提升生成速度，节省 API 成本。

### 2. 可以分模块重新生成内容
* **痛点场景**：生成的备课包整体很棒，但“文化背景”部分不够生动，或者“习题”部分难度太高。如果重新生成整个备课包，不仅耗时，还可能把之前满意的部分也改坏了。
* **产品落地点：修改【第三步：生成结果页】**
  * **设计方案**：摒弃目前一整块 Markdown 渲染的方案，改为 **“分块卡片式（Block-based）”** 布局。
  * 每一个生成的模块（如“词汇精讲”、“随堂测验”）都作为一个独立的卡片展示。
  * 每个卡片右上角提供三个操作：**“编辑（手动修改）”**、**“复制”**、**“重新生成本模块”**。
  * **价值**：实现局部精调，避免牵一发而动全身，大幅提升微调效率。

### 3. 可以手动调整提示词
* **痛点场景**：AI 的默认语气可能太死板，或者老师希望今天生成的教案是以“幽默风趣”的口吻，或者指定要针对“零基础”学生。
* **产品落地点：全局设置与局部微调结合**
  * **方案 A（全局预设）**：在页面顶部导航栏增加一个 ⚙️ **“AI 偏好设置”**。允许高级教师直接修改 System Prompt（系统提示词），例如预设角色、要求输出的格式等。
  * **方案 B（生成前干预）**：在第二步点击“生成”前，提供一个 **“附加要求（Optional）”** 的输入框。例如老师可以输入：“请侧重于动词变位的讲解，并加入一些关于法国大革命的背景”。
  * **方案 C（重生成时干预）**：在第三步点击某个模块的“重新生成”时，弹出一个小气泡，允许输入定向指令：“换一批更难的习题”或“用更通俗的语言解释这个语法”。

## 二、 从“一线教师视角”的深度头脑风暴

除了上述三点，站在初高中法语教师（面临升学压力、学情复杂、教务繁杂）的角度，我们的产品还可以在以下几个维度进行降维打击：

### 维度一：学情分层与差异化教学（Differentiation）
初高中存在明显的学情差异，AI 生成的内容如果难度一刀切，往往无法直接使用。
* **学生水平设定（CEFR 标准）**：在第一步输入素材时，提供目标学生水平下拉框（A1, A2, B1, B2 或 高考法语/中考法语）。AI 生成的例句词汇量、语法解释深度将严格限制在该级别。
* **习题难度分级**：不仅是例句，生成的随堂测验和课后作业也必须严格遵循所选的欧标等级。例如 A1 级别只出简单的单选和连线，B1 级别增加主观表达题和长句翻译。
* **分层作业生成**：在生成“课后作业”模块时，自动生成两套：A套（基础巩固），B套（拔高拓展），方便老师进行分层布置。

### 维度二：输出格式与实操落地（Output & Export）
Markdown 格式在网页上好看，但老师最终上课用的是 PPT，发给学生的是打印纸。
* **教师版 vs 学生版（挖空版）导出**：这是**刚需中的刚需**！老师上课时手里拿的是“答案版”，发给学生的是“填空版”。在第三步，应该提供一键将讲义中的关键生词、动词变位“挖空”生成学生讲义的功能。
* **一键生成 PPT 大纲**：自动将长文本提炼为短句（Bullet points），并导出为结构化的格式，老师可以直接贴进 PPT 甚至一键生成 PPT 文件。

### 维度三：消除 AI 幻觉的进一步保障（Safety & Accuracy）
尽管我们做了分步大纲确认，但具体内容（如例句）依然可能出错，语言教学容错率极低。
* **双语对照审查**：在生成的教案中，所有法语例句默认附带精准的中文翻译，帮助老师一眼扫出语义是否有偏差。
* **词典外链/词根词缀溯源**：鼠标悬停在 AI 生成的高级法语词汇上，能够自动弹出权威词典（如拉鲁斯）的解释，方便老师备课时自我充电和核实。

### 维度四：资产沉淀与协作（Asset Management）
* **“我的备课本”云端存档**：目前页面一刷新，备课包就没了。需要增加简单的本地存储（Local Storage）或账号系统，保存老师的历史备课记录。
* **自建语料库/题库**：如果老师觉得某次生成的题目特别好，可以点击“收藏到我的题库”。未来系统会越来越懂这位老师的出题风格。

## 三、 推荐的 v2.0 迭代优先级 (Roadmap)

考虑到开发成本与教师体验的 ROI（投资回报率），建议按以下顺序推进：

**🚀 P0 (必做，核心痛点，建议本周/下周迭代)：**
1. 第二步的**模块化 Checkbox 勾选**功能（词汇、语法、习题等）。
2. 第三步的**卡片式分块展示**与**单模块重新生成**功能。
3. 第一/二步增加**“附加指令 (Custom Prompt)”**和**“欧标难度选择(A1-B2)”**，且难度分级必须作用于习题生成。

**🌟 P1 (高优，显著提升产品专业感与壁垒)：**
1. 真实且格式完美的 **Word (.docx) 导出**功能。
2. “学生挖空版”讲义生成功能。

**🛠️ P2 (后续演进，锦上添花)：**
1. 历史记录保存。
2. 一键导出 PPT。
3. 全局高级 System Prompt 设置面板。
